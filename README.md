 # 图像分类网络  
 
 ## 运行环境
* Anaconda3（建议使用）
* python3.6/3.7/3.8
* pycharm (IDE)
* pytorch 1.7.1 (pip package)
* torchvision 0.8.1 (pip package)
* numpy 1.19.5 (pip package)
* matplotlib 3.3.4 (pip package)
* tqdm 4.48.0 (pip package)

 ## 训练自己的数据集
 
 1. 根据要求整理dataset
 2. 用alexnet、vggnet（或者其他网络）中的train.py进行模型训练
